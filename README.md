# Personal Finance Dashboard

A clean, modular personal finance tracking application built with React.js using a feature-based architecture pattern.

## Features
- **Global Overview**: Real-time stats on your budget, accrued spending, and available balance.
- **Log Management**: Store and track different types of financial outgoings.
- **Responsive Layout**: Designed for seamless use on any device configuration.
- **Dynamic State**: Utilizes the React Context API with Reducers to orchestrate robust state changes across modules.

## Architecture & Tech
- **Core Engine**: React.js 18 (Hooks, Context, Reducers)
- **Styling Layer**: Advanced Bootstrap 5 Overrides + Custom CSS Variables
- **Component Organization**: Segmented into Modules (`Dashboard`, `Transactions`, `Layout`) and Stores correctly separated instead of a flat component tree.
- **Icons**: React-Icons (`TiDelete`)

## Getting Started

1. Clone repository to your local machine.
2. Install node runtimes using `npm i` or `yarn install`.
3. Launch via `npm start`.
